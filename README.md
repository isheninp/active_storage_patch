# ActiveStoragePatch
Removes 302 redirects

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'active_storage_patch', github: 'isheninp/active_storage_patch'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install active_storage_patch
```
